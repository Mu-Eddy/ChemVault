deployment posture

Privacy-first static architecture

This build uses local JavaScript data files and browser storage instead of external APIs. A future onion deployment can serve the same static assets behind Tor, then add a small backend only for encrypted submissions.

ChemVault is for legal education, revision and lab-report support. It is not a replacement for SDS documents, COSHH forms, supervisor instructions or local regulations.
