# ChemVault

ChemVault is an experimental chemistry-focused research portal and digital archive built using static web technologies and deployed globally through Cloudflare Workers.

The project combines:

* Research dossier systems
* Chemical data visualization
* Interactive laboratory archives
* Experimental interfaces
* Cyberpunk-inspired scientific UI design

## Live Website

https://home.chemvault.science

---

# Features

## Research Desk

Interactive research interface for navigating:

* Chemical records
* Experimental archives
* Molecular data
* Scientific notes

---

## Dossier System

Dynamic dossier-style panels for displaying:

* Research subjects
* Chemical classifications
* Experimental metadata
* Hazard annotations

---

## Interactive Frontend

Built using:

* HTML5
* CSS3
* Vanilla JavaScript

Includes:

* Animated interfaces
* Dynamic loading systems
* Interactive panels
* Multi-layer UI effects

---

## Cloudflare Deployment

The project is deployed using:

* Cloudflare Workers
* GitHub integration
* Automatic CI/CD deployment

All updates pushed to the `main` branch are automatically deployed worldwide.

---

# Deployment

This project uses automatic deployment through Cloudflare.

## Workflow

```plaintext
Local Edit
↓
Push to GitHub
↓
Cloudflare detects changes
↓
Automatic deployment
↓
Website updated globally
```

# Security

The project uses Cloudflare security features including:

* HTTPS enforcement
* Bot protection
* Browser integrity checks
* Static asset isolation

No backend database or server-side runtime is currently exposed.
